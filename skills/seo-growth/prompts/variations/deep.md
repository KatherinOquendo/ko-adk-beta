# Variation — Deep (seo-growth)

`depth = deep`. Apply one playbook exhaustively, verifying at each step.

## When to use
The user wants the full deliverable for one topic: a complete technical-SEO
implementation, a whole funnel blueprint with scoring + nurture, a full landing page, or a
rigorous CRO program for a page.

## Exhaustive loop
1. Resolve the one topic; confirm scope boundaries (what is in vs out per the playbook).
2. Read that one playbook in full.
3. **Discover** every input the playbook lists; instrument and capture any missing baseline
   before proceeding — do not test or claim on absent data.
4. **Analyze** with the playbook's full decision apparatus:
   - `seo-architecture` → rendering per page type, full canonical/hreflang/redirect map.
   - `seo-content` → keyword map (primary + secondary), schema per page type, full heading tree.
   - `conversion-optimization` → classify every friction point, ICE-rank all, then act on top.
   - `funnel-design` → all three stages, full lead-scoring model with negative scoring,
     complete nurture flow with terminating exits, deterministic handoff.
5. **Execute** the complete artifact, script-first where a deterministic script exists.
6. **Validate** against *every* item in the playbook's Quality Criteria; list each as
   pass/fail with evidence, not a blanket "looks good."

## Constraints
Evidence-tag every non-obvious claim; never fabricate metrics; record trade-offs explicitly;
a flat/negative experiment result is a kept learning. Single brand, no prices, no PII.
