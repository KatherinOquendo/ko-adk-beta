# Quick variation — task-subfolder

Fast path for a clear, single goal with no special constraints.

1. Goal present? If empty → `{VACIO_CRITICO}`, stop.
2. `id = max(T-NNN)+1`; `slug` from goal (lowercase/ASCII/hyphens).
3. Write the three files **missing-only**:
   - `CLAUDE.md` — scope, allowed tools, tag family, "read log.md first", parent
     link, log ordering stated once.
   - `task.md` — goal `{EXTRAIDO_HILO}` + a short acceptance checklist
     (`{POR_CONFIRMAR}` if drafted).
   - `log.md` — `## YYYY-MM-DD · session 1 — created` + one-line status.
4. Run the gate (SKILL.md §4). Summary: id, path, written/skipped.

Skip nothing in the gate. No prices, single brand, Jarvis tags only.
