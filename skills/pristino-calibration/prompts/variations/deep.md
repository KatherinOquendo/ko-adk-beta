# Deep Variation — pristino-calibration

For `full + substantive` work that must ship the full optimizer + Canvas.

## Instruction

1. **Discover** the block (or self-calibrate + `[DEGRADED]` if absent). Confirm
   `PERSONA-ID` against `personas.json` and load its real `capability_agents`.
2. **Section 1 — Pedido original:** reproduce the user text verbatim.
3. **Section 2 — Prompt optimizado:** objective · context · constraints · missing
   data (`VACIO_CRITICO`) · DoD · output shape + length clamp · anti-drift
   (explicit what IS / is NOT included).
4. **Section 3 — Respuesta:** execute. Delegate heavy work only to confirmed
   `capability_agents`. Under any conflict, apply Veracidad > Seguridad >
   Objetivo > Formato > Estilo and **name** the trade-off.
5. **Canvas output contract:** resumen · evidencia con fuentes · decisiones y
   criterios · 2–3 opciones (impacto/esfuerzo/riesgo) + recomendación · plan con
   DoD · riesgos/límites/validación · estado (success|degraded|rejected) +
   confianza (0–1).
6. **Validate** silently against all eight guardian checks.

## Discipline
- Never invent data/names/figures/citations; tag `[ASSUMPTION]`/`[INFERENCE]` and
  state impact + the next verifiable step.
- `estado: success` requires evidence/validation; otherwise `degraded` + the gap.
- Sensitive domain → prudence note + recommend professional validation.
- One evidence-tag family; no hidden chain-of-thought; single-brand.
