# Prompt — variation: quick (qbr-quarterly)

QBR comprimido para un trimestre con pocos OKRs (<=4) y evidencia ya a mano. Misma
disciplina, menos profundidad.

## Pasos
1. **Discover express** — confirma baseline + evidencia en una pasada. Sin baseline →
   `{VACIO_CRITICO}`, para y pide.
2. **Audit** — tabla rapida: meta | objetivo | observado | estado | tag de fuente.
   Sin metrica → `{POR_CONFIRMAR}`.
3. **Learn** — 2-3 lecciones a causa raiz, una linea cada una.
4. **Plan** — 3 objetivos del proximo Q, cada uno con metrica + owner.
5. **Validate** — checklist minimo: estados con evidencia, lecciones conectadas,
   objetivos medibles, una sola familia de tags.

## Salida
Version corta de `templates/output.md`: Scorecard (tabla), Lecciones (bullets), Plan
(3 objetivos), Riesgos cross-quarter (si los hay).

## No negociable
Una sola familia de tags `{...}`; verde nunca por defecto; sin precios inventados.
