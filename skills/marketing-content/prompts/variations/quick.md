# Quick Variation — marketing-content

For `depth=quick`: route fast, execute essentials, gate the non-negotiables.

1. Resolve `topic` in one pass. Only ask if genuinely ambiguous between two.
2. Load the one playbook. Confirm brand.
3. Run Discover → Execute on the playbook's core structure only (skip exhaustive
   option analysis). Tag every claim as you draft.
4. Gate the non-negotiables for the topic:
   - copywriting → one framework, single CTA named in closing beat.
   - case-study → before/after hero metric + one approved verbatim quote.
   - press-release → lede ≤35 words + dateline + `###`.
   - video-script → hook <3s + one-verb CTA + pace budget.
   - whitepaper → single thesis stated + citations resolve.
   - content-calendar → pillars + owner/metric per slot.
   - podcast-prep → tiered questions + fact-check flags.
   - event-marketing → one CTA per asset.
5. Always enforce: tags, no invented prices, no unsourced superlatives,
   single-brand, no PII.

Output: routing block + artifact + a short pass/fail on the non-negotiables.
