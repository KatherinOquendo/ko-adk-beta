# Press Release
> "Method over hacks."
## TL;DR
Inverted pyramid, quotes, boilerplate, distribution strategy. [EXPLICIT]

## Anti-scope
- Not a blog post, not ad copy, not a pitch deck. PR = newsworthy fact + third-party-quotable framing. [EXPLICIT]
- No prices, projections, or superlatives ("best", "leading") without an attributable source. [EXPLICIT]

## Procedure
### Step 1: Discover
- Capture the 5 W's + 1 H (who/what/when/where/why/how) and the single newsworthy hook. [EXPLICIT]
- Confirm: dateline (CITY, Date), embargo/release date, approved spokesperson + title, distribution targets. [EXPLICIT]
### Step 2: Analyze
- Rank facts by news value; the lede must answer what + why-it-matters in <=35 words. Evaluate angle per Constitution XIII/XIV. [EXPLICIT]
- Draft 1-2 quotes that state opinion/vision (never restate facts the body already gives). [EXPLICIT]
### Step 3: Execute — inverted pyramid
1. **Headline** <=12 words, active voice, no period. **Subhead** (optional) adds one supporting fact. [EXPLICIT]
2. **Lede** — dateline + most important fact first. [EXPLICIT]
3. **Body** — supporting facts in descending importance; one quote near the top. [EXPLICIT]
4. **Boilerplate** — "About [Org]" evergreen paragraph, reused verbatim across releases. [EXPLICIT]
5. **Contact** — name, email, phone. Close with `###` (end marker). [EXPLICIT]
### Step 4: Validate
- Read top-down; if any paragraph could be cut from the bottom without losing the story, the order is correct. [EXPLICIT]
- Verify every claim is sourced/quotable and tags applied. [EXPLICIT]

## Worked example (lede + quote)
> SEATTLE, June 11, 2026 — Acme Corp today launched Orbit, an API that cuts integration time from weeks to hours.
> "Orbit removes the last manual step our customers dreaded," said Dana Lee, CTO. [EXPLICIT]

## Quality Criteria
- [ ] Headline <=12 words, active voice, no terminal period
- [ ] Lede answers what + why in <=35 words, dateline present
- [ ] >=1 attributed quote that adds opinion, not repetition
- [ ] Boilerplate + contact block + `###` end marker present
- [ ] Evidence tags applied; Constitution-compliant; no unsourced superlatives/prices

## Usage

Example invocations:

- "/press-release" — Run the full press release workflow

## Assumptions & Limits

- Assumes access to project artifacts (code, docs, configs) and an approved spokesperson quote/title [EXPLICIT]
- Requires English-language output unless otherwise specified [EXPLICIT]
- Does not replace legal/comms sign-off or domain expert judgment for final decisions [EXPLICIT]

## Edge Cases

| Scenario | Handling |
|----------|----------|
| Empty or minimal input | Request clarification before proceeding |
| Conflicting requirements | Flag conflicts explicitly, propose resolution |
| Out-of-scope request | Redirect to appropriate skill or escalate |
| No approved quote/spokesperson | Draft placeholder quote, flag `[NEEDS APPROVAL]`, do not publish |
| Embargo specified | Mark "EMBARGOED UNTIL <date/time TZ>" at top; never distribute early |
| Unsourced superlative/metric | Strip or attribute to a named source before release |
