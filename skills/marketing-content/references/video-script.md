# Video Script
> "Method over hacks."
## TL;DR
Structure short-form/explainer video: hook-body-CTA, per-beat timing, B-roll cues, captions, accessibility. Output is a two-column shot list (VISUAL | VO/ON-SCREEN), not prose. [EXPLICIT]
## Procedure
### Step 1: Discover
- Capture: platform + target duration, single CTA, audience, one core message (if >1, split into multiple scripts) [EXPLICIT]
### Step 2: Analyze
- Pick hook pattern; budget seconds per beat (see Timing); evaluate per Constitution XIII/XIV
### Step 3: Execute
- Write two columns: VISUAL (shot/B-roll) | VO + ON-SCREEN TEXT. Tag claims with evidence tags. Mark B-roll `[B-ROLL: ...]`, captions `[CC]`
### Step 4: Validate
- Read VO aloud against the clock; confirm hook lands <3s and CTA is one verb
## Structure & Timing
Hook 0–3s · Body (2–4 beats, one idea each) · CTA last 3–5s. Budget by total length:
| Length | Hook | Body | CTA |
|--------|------|------|-----|
| 30s | 0–3s | 22s | 5s |
| 60s | 0–3s | ~50s | 5s |
| 90s+ | 0–5s | rest | 5–8s |
- Spoken pace ≈ 2.5 words/sec → 60s ≈ 150 words max [EXPLICIT]
## Hook Patterns
Question · Bold claim · Pattern-interrupt · Stat · "Stop doing X". Hook must state stakes, not greet ("Hey guys" is banned). [EXPLICIT]
## Craft Rules
- One CTA, one verb, on-screen + spoken (e.g. "Comment ROUTER")
- B-roll every 3–5s to avoid static frames; cue each explicitly `[B-ROLL: ...]`
- Open captions `[CC]` always — 85% of feed views are muted [EXPLICIT]
- VO sentences ≤12 words; cut adjectives before nouns
## Worked Example (15s, vertical, muted-first)
| VISUAL | VO / ON-SCREEN |
|--------|----------------|
| Close-up, fast zoom `[B-ROLL: cluttered desk]` | "Your scripts ramble because you skip the clock." `[CC]` |
| Split-screen timer overlay | "Hook in 3 seconds. One idea per beat." |
| Card: "1 CTA = 1 verb" | "End with a single action." |
| Logo + comment prompt | "Comment ROUTER for the template." `[CC]` |
## Quality Criteria
- [ ] Hook conveys stakes within 3s; no greeting filler
- [ ] One core message, one CTA (single verb)
- [ ] Word count ≤ pace budget for stated length
- [ ] B-roll cue + caption marked on every beat
- [ ] Evidence tags applied; Constitution-compliant

## Assumptions & Limits

- Assumes platform, target duration, and one CTA are known or askable [EXPLICIT]
- English-language output unless otherwise specified [EXPLICIT]
- Scope: scripting only — not editing, shot capture, thumbnails, or publishing [EXPLICIT]
- Does not replace domain expert judgment for final decisions [EXPLICIT]

## Edge Cases

| Scenario | Handling |
|----------|----------|
| Empty or minimal input | Request platform + duration + CTA before proceeding |
| Multiple core messages | Split into separate scripts; do not pack one video |
| No duration given | Default to 60s, state the assumption [SUPUESTO] |
| Conflicting requirements | Flag conflicts explicitly, propose resolution |
| Out-of-scope request | Redirect to appropriate skill or escalate |
