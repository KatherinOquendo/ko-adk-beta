# Prompt — Quick (wbr-weekly-review)

WBR express para una semana de bajo volumen o un alcance pequeno. Mantiene el rigor
del gate pero recorta el diagnostico a lo esencial.

## Entrada minima
- Semana ISO + alcance unico.
- Compromisos previos (estado) y senales clave de la semana.

Si falta periodo o alcance → `{VACIO_CRITICO}`, pregunta y detente.

## Pasos
1. Lista avances cerrados con evidencia (bullets verbo-en-pasado).
2. Lista estancados con una linea cada uno: `item — dueno — proximo paso (fecha)`.
3. Marca friccion en una linea: recurrente o puntual.
4. Deriva 1-3 compromisos con criterio de hecho.
5. Corre el gate corto: dueno+fecha en estancados, compromisos previos marcados,
   una sola familia de tags `{...}`, nada verde sin evidencia.

## Salida
Acta WBR compacta (puede omitir la tabla completa y usar bullets), pero **sin
omitir** dueno, proximo paso fechado ni compromisos previos incumplidos. Never
green-as-success. {DOC}
